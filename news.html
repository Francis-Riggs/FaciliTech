Elevation SRS 1.15 Update Has Been Released

October 7, 2019

Today, update 1.15 for Elevation SRS has been released. This update introduces a plethora of tweaks and bug fixes as well as a few new features. Users and operators are now able to receive text message notifications for work order updates, which can be enabled or disabled from the account settings. In addition, users can now attach up to 5 files to each work order, and the maximum file attachment size has been increased from 5 MB to 10 MB. For a full list of changes, please consult the changelog.
